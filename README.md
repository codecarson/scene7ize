# Scene7izer

Converts local image paths in an HTML or CSS file to Scene7 paths containing dimensions and format info.

## Installation

Add this line to your application's Gemfile:

    gem 'scene7izer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install scene7izer
